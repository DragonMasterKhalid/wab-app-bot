# Telegram Web Bot Starter

Quick start:
1. copy repository
2. cd server
3. cp .env.example .env  and fill BOT_TOKEN, HOST (eg https://yourdomain.com), ADMIN_TOKEN
4. npm install
5. npm run dev   (or `npm start`)

Visit:
- User panel: https://yourhost/user?user_id=12345   (Telegram web_app will open this)
- Admin panel: https://yourhost/admin?token=supersecretadmintoken
